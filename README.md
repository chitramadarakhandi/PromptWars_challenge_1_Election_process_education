# VoteWise – Election Education Dashboard (Multi-Feature)

VoteWise is a comprehensive, modern web application designed to educate and guide voters through the election process via interactive tools. 

## Platform Features

1. **Dashboard Home**: Centralized view of smart insights and quick action buttons.
2. **Personalized Voting Plan**: Generates custom roadmaps based on age and voter history.
3. **Voting Simulator**: Allows users to practice digital ballot casting safely online.
4. **Interactive Checklist**: Provides actionable, trackable steps ensuring voting readiness.
5. **Eligibility Checker**: Computes basic prerequisites for valid voting capability.
6. **Timeline Visualizer**: Standard timeline UI charting election seasons.
7. **AI Chat Assistant**: A secondary fallback layer powered by Gemini to answer edge-case queries.

## Google Services Integrated

- **Google Gemini API**: Native AI chatbot interface to solve ad-hoc user questioning.
- **Firebase Firestore & Hosting**: Recommended setup for logging user trends (insights) and easy lightweight deployment.
- **Google Analytics**: Placeholder installed in root HTML.

## Setup

1. Open `index.html` in any modern web browser.
2. (Optional) Provide your API key in `script.js` to enable live Chat.
